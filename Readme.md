遙控板　RemoteControl Mini
========
* 控制器　STM32F407V 100Pin 168MHz DSP FPU
* 螢幕　3.5吋螢幕480*320，使用R61581，FSMC操作
* 儲存紀錄　SD卡，使用SDIO操作
* 乙太網路　W5500，使用SPI操作
* 感測器　IMU 6-DOF (MPU-6050)
* 無線傳輸　nRF24L01P + PA + LNA
* 1個USB接口　連接電腦、外擴裝置
* 1個UART接口　外擴裝置

Hardware
========
* Use Altium Designer 13
* PCB Size : 155 * 60mm
