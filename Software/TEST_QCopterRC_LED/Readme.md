QCopterRC LED Example
========
* Author  : [Hom](https://github.com/Hom19910422)
* Update  : 2014/01/01

Description
========
QCopterRC LED 和 KEY 的範例程式，啟動時 LED_Y 會閃爍，當按下不同按鍵組合時，LED 會有不同的亮暗，放開按鍵 LED_Y 又會持續閃爍。

Hardware
========
* KEY  
PA0　- KEY_LL  
PA1　- KEY_LP  
PA8　- KEY_RP  
PB2　- KEY_LR  
PD3　- KEY_RL  
PD6　- KEY_RR  
PE0　- KEY_PR  
PE1　- KEY_PP  
PE2　- KEY_PL  
PE3　- KEY_S1  
PE4　- KEY_S2  
PE5　- KEY_S3  

* LED  
PC13 - LED_B  
PC14 - LED_G  
PC15 - LED_R  
PC15 - LED_R  
PE6　- LED_Y
